Automation Creation Orthotics
📖 Descrição
Este projeto tem como objetivo automatizar processos relacionados ao desenvolvimento e personalização de órteses. Ele busca integrar ferramentas digitais para otimizar a criação, reduzir erros manuais e acelerar o fluxo de trabalho em clínicas, laboratórios e indústrias de dispositivos médicos.

Funcionalidades
- Automação de etapas de modelagem de órteses.
- Integração com softwares CAD/CAM.
- Geração de relatórios e documentação técnica.
- Suporte a diferentes tipos de dispositivos ortopédicos.

Instalação
Clone o repositório:
git clone https://github.com/LOP3D-Unifesp/automation_creation_orthotics.git
cd automation_creation_orthotics

Instalação no Blender version > 3.6
Configurar o add-on:
- Os add-ons devem estar em formato .zip (precisa compactar).

Abrir o Blender
- Vá em Edit → Preferences (Editar → Preferências).
- Instalar o add-on
- Na aba lateral, clique em Add-ons.
- Clique em Install… (Instalar).
- Selecione o arquivo .zip do add-on.
- O Blender vai carregar o add-on automaticamente.
- Ativar o add-on

Depois de instalado, o add-on aparece na lista.
- Marque a caixa de seleção ao lado do nome para ativá-lo.
- Agora ele estará disponível no Blender (geralmente em menus específicos ou no painel lateral “N”).
- Salvar preferências
- Clique em Save Preferences para que o add-on continue ativo nas próximas vezes que abrir o Blender.
